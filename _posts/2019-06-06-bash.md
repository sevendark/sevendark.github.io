```sh
$ echo "asda" > bbb.txt
```
# 常用命令

## grep

```sh
-x, 匹配整行
-f, 从文件中获取匹配值
-F, 使用换行分割匹配值
-v, 反向过滤
```

```sh
$ grep -x -F -f aaa.txt bbb.txt
asda
$ grep -x -F -f bbb.txt aaa.txt
asda
$ grep -x -F -v -f bbb.txt aaa.txt
asd
```

## diff

```sh
$ diff -c aaa.txt bbb.txt
```
